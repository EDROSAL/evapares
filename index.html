Integración con Google Sheets (opcional)

Sin configurar esto, los datos se guardan en este navegador. Con la integración, cada respuesta se envía también a tu Google Sheet.

Instrucciones

  1. Ve a Google Sheets → crea una hoja nueva para este curso.
  2. Abre Extensiones → Apps Script.
  3. Borra el código existente y pega el código de abajo.
  4. Clic en Implementar → Nueva implementación → Aplicación web.
  5. En "Quién tiene acceso" elige Cualquier usuario.
  6. Copia la URL resultante y pégala arriba.
